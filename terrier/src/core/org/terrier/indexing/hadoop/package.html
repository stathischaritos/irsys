Provides classes for Terrier's MapReduce indexer. In particular, the indexer classes are here. The application class is HadoopIndexing, while the low-level classes are in org.terrier.structures.indexing.singlepass.hadoop.

The classes in this package are:

  • Hadoop_BasicSinglePassIndexer - a distributed singlepass indexer that makes use of a Hadoop MapReduce cluster.
  • Hadoop_BasicSinglePassIndexer - as Hadoop_BasicSinglePassIndexer, but also records position information.
